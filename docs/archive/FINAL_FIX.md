# ✅ Финальное исправление проблемы с логином

## Проблема
Прокси Vite не работал - запросы шли напрямую на Apache и получали 404.

## Решение
Использовать прямой URL на `https://planrun.ru/planrun` вместо прокси.

### Изменения в `ApiClient.js`:
- `baseUrl` по умолчанию теперь `'https://planrun.ru/planrun'`
- Все запросы идут напрямую на сервер без прокси
- CORS заголовки уже настроены в `login.php` и `api.php`

## Преимущества
- ✅ Нет проблем с прокси
- ✅ Нет редиректов
- ✅ Работает напрямую через HTTPS
- ✅ CORS настроен правильно

## Как проверить
1. Обновите страницу в браузере (Ctrl+F5)
2. Попробуйте залогиниться
3. Проверьте Network tab - запросы должны идти на `https://planrun.ru/planrun/login.php`
