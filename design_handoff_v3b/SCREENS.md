# PlanRun v3B «Телеметрия» — спецификация экранов для Claude Code

Каждый экран из `mocks/` ↔ файлы реального кода. Моки — **каноничная вёрстка** (инлайн-стили читаются как спека). Колонка «Код портала» — откуда брать ЛОГИКУ (и что затем удалить).

Файлы моков: r3-shared.jsx (данные+примитивы), r3-b.jsx (дизайн-система+дашборды), r3b-calendar / r3b-workout / r3b-chat / r3b-extra (статистика+онбординг) / r3b-settings / r3b-social (уведомления+тренеры+профили) / r3b-coach2 (drill-in+мастер+шаблоны) / r3b-extra2 (месяц+статистика-деск) / r3b-landing (+auth) / r3b-final (кабинет тренера) / r3b-admin (+пароли+privacy) / r3b-modals / r3b-flow (карта переходов).

## Навигация (обязательно к реализации)
- **Таб-бар бегуна** (BottomNav.jsx): Главная `/` · План `/calendar` · Чат `/chat` · Прогресс `/stats` · **Профиль** (страница пользователя). Активная вкладка — градиентная pill.
- **Страница пользователя** (мок `BAthleteProfileMobile`): аватар + имя, чип режима, цель с кольцом, рекорды, лента тренировок, «Мои тренеры», **шестерёнка → /settings**. Drawer-меню отменён.
- **Каталог тренеров доступен** из: профиль → «Мои тренеры»; режим-чип на главной → «Живой тренер»; лендинг → «Смотреть тренеров».
- Полная карта — артборд «Карта переходов» (мок `BFlowMap`).
